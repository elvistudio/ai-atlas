# Contributing to AI Atlas

Thank you for helping improve AI Atlas.

## Current stage

The project is currently at:

> v0.1 — Clean hierarchy draft, Level 1 and Level 2 only.

## How to contribute

Please suggest changes through issues or pull requests.

Good contributions:

- suggest a missing Level 1 branch,
- suggest a missing Level 2 concept,
- explain why a concept should move,
- identify overlap or ambiguity,
- propose clearer names.

## What to avoid for now

Please do not add:

- Level 3 concepts yet,
- specific product versions,
- model version numbers,
- company-specific marketing names unless they are needed as examples,
- long explanations inside the taxonomy file.

## Taxonomy change format

When proposing a change, explain:

1. Current location
2. Proposed location
3. Reason for the change
4. Whether this affects Level 1 or Level 2

## Preferred style

Keep contributions small and reviewable.

Prefer one conceptual change per pull request.

If a change affects the canonical taxonomy JSON, make sure the human-readable Markdown version remains aligned.
